

# BookStore / Management System

A modern and responsive **ASP.NET Core MVC-based Management System** designed to handle business operations efficiently with a clean UI and structured backend.

---

## 📌 Features

* 📊 Dashboard Overview
* 📦 Inventory / Stock Management
* 🧾 Billing & Invoice System
* 👥 Customer / Party Management
* 📈 Reports & Data Tracking
* 🔐 Secure Authentication System

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, Bootstrap
* **Backend:** ASP.NET Core MVC
* **Database:** SQL Server
* **Architecture:** MVC Pattern

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone (https://github.com/MuhammadSaadfancy/bookstore.git)
```

2. Open project in **Visual Studio**

3. Configure database in:

```bash
appsettings.json
```

4. Run the project:

```bash
Ctrl + F5
```
